# CS 513 Final Project

We will be analyzing this dataset of Exoplanets: https://www.kaggle.com/datasets/kanchana1990/nasa-exoplanet-archive-intelligence

## Installing + Running

For the sake of running smoothly, I used conda to create a virtual environment for this project. The config file can be found at `/config/environment.yml`. To get up and running, go to the terminal and enter `conda env create -f environment.yml` (assuming you are using linux, there is a way to do this with windows and mac). In VS code select that environment as the kernel, and you should be good to go!
